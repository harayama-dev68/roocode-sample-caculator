# Caculator

React + Vite + Tailwind を使ったシンプルなブラウザ電卓アプリケーションの雛形です。

## セットアップ

1. 依存関係をインストール

```bash
npm install
```

2. 開発サーバー起動

```bash
npm run dev
```

3. ビルド

```bash
npm run build
```

## 設計方針
- モダンな開発体験（Vite + React）を優先
- UI は Tailwind CSS で高速に実装
- 計算ロジックは簡易な入力サニタイズ後に評価（小規模デモ用途）
- キーボード操作、アクセシビリティ考慮
